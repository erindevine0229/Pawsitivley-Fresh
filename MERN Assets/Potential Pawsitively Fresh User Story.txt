Potential Pawsitively Fresh User Story
* as a USER who is a pet owner I want to find food that meets the dietary needs of my dog/Cat.

* as a USER I want to be presented with options based on my input that can give me selections of food.

* as a USER I want to then be able to purchase the suggested foods.



Acceptance Criteria

* User is presented with a UX/UI

* User is able to create an account

* User is able to store that information for:
	login
	logout

* User is then present with options about their pet
	Type: Dog or Cat
	Size: large, medium, small
	Dietary Needs: An array that has preselected ingredients that may cause issues for pet [chicken, rice, turkey, corn, ect].

* User is then take to the list of foods that will meet those dietary requirements.
